# Makefile Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a self-documenting `Makefile` at the project root that lets a new contributor get Groceror running with a single `make setup` and discover every other command via `make help`.

**Architecture:** Single `Makefile` at the repo root. All tool paths are derived from a `VENV` variable so no system-wide installs are assumed. The `help` target (the default) is auto-generated from `##` inline comments via an `awk` one-liner.

**Tech Stack:** GNU Make, Python 3.12 venv, pytest, ruff, black, isort, uvicorn

---

### Task 1: Create the Makefile with variables, `.PHONY`, and `help`

**Files:**
- Create: `Makefile`

- [ ] **Step 1: Create `Makefile` with variables and the `help` target**

```makefile
VENV   = venv
PYTHON = $(VENV)/bin/python
PYTEST = $(VENV)/bin/pytest
RUFF   = $(VENV)/bin/ruff
BLACK  = $(VENV)/bin/black
ISORT  = $(VENV)/bin/isort
PORT   = 8000

.DEFAULT_GOAL := help

.PHONY: help setup run test test-unit test-integration lint format clean

help: ## Show available targets
	@awk 'BEGIN {FS = ":.*##"; printf "Usage:\n  make \033[36m<target>\033[0m\n\nTargets:\n"} \
	     /^[a-zA-Z_-]+:.*?##/ { printf "  \033[36m%-20s\033[0m %s\n", $$1, $$2 }' \
	     $(MAKEFILE_LIST)
```

- [ ] **Step 2: Verify `make help` prints the targets table**

Run:
```bash
make help
```
Expected output (colours stripped):
```
Usage:
  make <target>

Targets:
  help                 Show available targets
```
Only `help` appears at this point — that's correct.

### Task 2: Add `setup` and `run`

**Files:**
- Modify: `Makefile`

- [ ] **Step 1: Append `setup` and `run` targets**

Add after the `help` target:

```makefile
setup: ## Create venv and install all dependencies (run this first)
	python3 -m venv $(VENV)
	$(PYTHON) -m pip install --upgrade pip
	$(PYTHON) -m pip install -r requirements.txt
	$(PYTHON) -m pip install "pytest==9.0.3" "pytest-asyncio==1.3.0" "httpx==0.27.2"

run: ## Start the FastAPI dev server (override port: make run PORT=9000)
	$(PYTHON) -m uvicorn main:app --host 0.0.0.0 --port $(PORT) --reload
```

### Task 4: Add `lint`, `format`, `clean`

**Files:**
- Modify: `Makefile`

- [ ] **Step 1: Append lint, format, and clean targets**

```makefile
lint: ## Check code style (ruff + black + isort) — read-only, safe for CI
	$(RUFF) check .
	$(BLACK) --check .
	$(ISORT) --check-only .

format: ## Auto-fix code style (ruff + black + isort) — modifies files
	$(RUFF) check --fix .
	$(BLACK) .
	$(ISORT) .

clean: ## Remove __pycache__, .pytest_cache, .ruff_cache, and *.pyc files
	find . -type d -name __pycache__ -not -path "./.git/*" -exec rm -rf {} +
	find . -type d -name .pytest_cache -not -path "./.git/*" -exec rm -rf {} +
	find . -type d -name .ruff_cache -not -path "./.git/*" -exec rm -rf {} +
	find . -type f -name "*.pyc" -not -path "./.git/*" -delete
```
